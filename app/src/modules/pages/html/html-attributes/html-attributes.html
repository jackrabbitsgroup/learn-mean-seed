HTML Attributes
{{pagesFullPath + 'html/html-attributes'}}
HTML elements may be given attributes (also sometimes called properties) to specify additional information about that element. (AngularJS directives are also frequently written as attributes, using exactly the same syntax, but that's a lesson for another time.)
There are many attributes, some of which are specific to certain element types, but all attributes are coded using the following syntax:

<element_name attribute_name = "attribute_value" > (Interior of element goes here) </element_name>
For example:
<div class = "my-class" > </div>

In the example above, the div is assigned a class called 'my-class'. 'class' is the name of an attribute that is recognized by div elements. 'my-class' is a value that we are assigning to this attribute. Attribute values in HTML always follow an '=' symbol and are always surrounded by a matching pair of quotation marks. Either single or double quotes may be used, provided that they match, meaning that the following is also correct:

<div class = 'my-class' > </div>

Attribute assignments are always made inside the element's opening tag, after the element name and before the closing >. Multiple attributes may be assigned to the same element as a space-separated list (you could even omit the spaces, actually, but that would be messy and harder to read):

<div class = 'my-class' style = 'color:red;' > </div>

Remember that not every attribute works with every element type ('class' and 'style' work with just about anything, but other attributes, like 'src' or 'href', work with only a select few elements), and attribute value syntax varies depending on the attribute. The more important attributes will be covered in more detail by these lessons, but should you ever have any additional questions, a quick internet search or two on a given html element will show you all of the attributes it accepts and what they're for.
