HTML: The Div Element

The 'div' element is easily the most common of all html elements. Divs are used to segment a page into pieces, which you can then move around and style using CSS.

This makes divs ideal for setting up your page's layout; you can wrap any piece of a page inside a div, and then move the piece about the page by manipulating the div. It's entirely possible (and in fact, quite common) to set up your page's basic layout using nothing but divs and CSS.

By default, a div is simply a rectangular box that spans the entire width of its container. Its height is determined by the height of whatever is placed inside it, and it will begin immediately below whatever element came before it. That's really all there is to it - the rest is CSS.

Protip: You can never have too many divs. Oftentimes, what seems like a difficult layout problem can become easy simply by wrapping the problem area(s) in an extra div or two. The div's ability to segment the page allows you isolate problems, separating them from interference from the rest of your html. It may help to think of each div as cordoning off a section of the screen, and within that cordoned-off space you are essentially working in a new (albeit smaller) blank page.
