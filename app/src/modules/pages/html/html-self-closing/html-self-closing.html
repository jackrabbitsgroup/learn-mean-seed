HTML: Self-Closing Elements
{{pagesFullPath + 'html/html-self-closing'}}
You should know by now that most html elements have an opening tag (like <div>) and a closing tag (</div>), which mark the start and end of the element, respectively. Some elements, however, do not follow this rule. They are called self-closing.

A self-closing element does not have a closing tag; instead, the opening tag closes itself, ending with />. Here is an example:

<br />

It is impossible to nest any other text or html inside of a self-closing element, because self-closing elements do not have an inside. They may have attributes defined in the opening tag just like other html elements, but there is no closing tag, and therefore no space to put anything inside the element.

Here is a quick list of self-closing elements that you should be aware of, along with a brief description of each:

<br /> Puts a blank line on the page, like a newline character.
<img src = 'path/to/image.jpg' /> Displays an image on the page.
<input type = 'text' /> Creates an input where a user can type or interact, like for a login form.

There are other self-closing elements, but you'll rarely, if ever, have to deal with them.
