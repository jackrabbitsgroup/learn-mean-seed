The a element, short for 'anchor', is used to create links to other pages. Its most important attribute is href, which tells the tag where to link to.

This is a link to the AngularJS home page.

The anchor tag creates a clickable block on the page; clicking that block directs the browser to the URL specified by the href. Anchors can contain more than just text, too - in fact, you can put just about anything inside of one. The image below is wrapped in an anchor tag that points to Google's home page.


The a element also has a target attribute, which specifies where to open the link. Setting target = '_blank' is the most common, and will cause the link to open in a new tab.

This is a link to the AngularJS home page. It will open in a new tab, so clicking here won't make you leave this page.

In Angular, we often use ng-href instead of href, because this allows us to safely use Angular variables in our links. This is particularly useful for creating links to other pages on your own website, and in Mean Seed we have a variable defined for you called appPathLink for precisely this purpose. Observe:

This is a link to this site's lesson on the div element. appPathLink takes care of the annoying beginning part of the link, so I only have to add the part that comes after the / in the url; in this case, html-div.

ng-href can do everything href can do, so if you're working in Angular there's no reason not to use it.
