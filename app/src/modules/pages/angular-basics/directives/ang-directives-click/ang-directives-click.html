Angular Basics: ng-click
{{pagesFullPath + 'angular-basics/directives/ang-directives-click'}}
As the name implies, ng-click is used to add a click event to an element. The syntax is very simple:

ng-click = 'insertJavascriptHere'

When the element with the ng-click on it is clicked (or tapped, in the case of touchscreens - ng-click handles both), the specified Javascript code will be executed. In this code, you have access to $scope variables and the $event, should you need it.

Typically, you'll simply reference a single $scope function, as in

ng-click = 'myScopeFunction("some_input")'

You can put almost any basic Javascript code there, though. You can even use multiple lines of code:

ng-click = 'myScopeFunction("some_input"); my_scope_variable = my_scope_variable + 1; my_other_scope_variable = false'

Sometimes it's very helpful to put multiple lines of code in a click event, but in general, try to keep them to a minimum - Javascript doesn't really belong in an HTML file.

Here are some examples. Make sure you understand them! Click events are the primary way that users interact with your page, after all.

Click to increase $scope.count1


Click to decrease $scope.count1 and increase $scope.count2


Click to add 5 to $scope.count1, subtract 2 from $scope.count2


$scope.count1: {{count1}}
$scope.count2: {{count2}}
