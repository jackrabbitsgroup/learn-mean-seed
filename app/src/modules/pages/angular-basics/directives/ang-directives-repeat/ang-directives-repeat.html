Angular Basics: ng-repeat

ng-repeat is one of the most useful directives Angular has to offer. It also tends to be the most confusing ng- directive for beginners.

ng-repeat is a bit like an HTML version of a Javascript for loop. You give this directive an array, and it creates an HTML element for every entry in the array. Here's an example - check this page's Ctrl file to see the array:

This is item {{$index}}.

{{item.text}}

ng-repeat creates a copy of the element it is placed on - one copy for every element in the specified array. Think of this element as a template: you can put any HTML you like inside it, and it will all be copied faithfully. More importantly, you can reference the individual element's information in the template, allowing you to make each copy a little bit different.

In the example above, we use ng-repeat = 'item in my_array'. my_array is a Javascript array defined in our Ctrl file. It has three elements, each an object with two properties, bg_color and text. item is just the variable name we're going to use to reference the current element; we could have just as easily said foo in my_array.

Each repeated element is given its own new $scope, and in this scope, item (or foo or whatever the case may be) holds the current element. That's why we can reference things like item.bg_color inside the repeated element as if it's a $scope variable.

ng-repeat also gives us $index, the index of the current element, as well as booleans $first, $last, $middle, $even, and $odd. These booleans are fairly self-explanatory - $first is true only for the first element, $middle is true for all but the first and last elements, $even is true if $index is an even number, and so on.

As you might have guessed by now, ng-repeat is absurdly useful. It allows you to very quickly create a template element, and then all you have to do is fill your array with entries and you'll get a copy of the template for each entry. And you can add, edit, or remove entries simply by adjusting the array in Javascript; since it's a $scope variable, everything will update automatically.
