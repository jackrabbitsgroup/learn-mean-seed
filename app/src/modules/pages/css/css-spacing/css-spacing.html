CSS Spacing: Margin, Border, and Padding
{{pagesFullPath + 'css/css-spacing'}}
Adding spacing between elements is one of the most common tasks in CSS, and luckily for us, it's not difficult. The three relevant properties are margin, border, and padding.

You already know what a border is - it surrounds the outside of an element, usually as a thin, solid line. Margin is the space on the outside of the border, separating the border from any elements surrouding our element. Padding is the space on the inside of the border, separating the border from the interior of the element. This image says it all:



Each of these properties has four sides: top, bottom, left, and right. You can define all four at once with something like padding: 10px;, or you can specify each individually using margin-top, margin-left, etc.
