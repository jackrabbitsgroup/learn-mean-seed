CSS Layout: Float
{{pagesFullPath + 'css/layout/css-layout-float'}}
There are three common ways to cause div elements to display side-by-side: Flexbox , Inline-Block , and float.

Setting float: left (or float: right) causes a div's width to shrink to the width of its content, and to display as far to the left (or to the right) as it can. Elements after a floated element will flow around it, until the float is cleared. If a floated element cannot fit on the current line, it will wrap to the next line.

Floats should always be cleared, otherwise they can alter the display of everything that follows after them. This can be done by adding the style clear:both to an element after the floats. In Mean Seed, a float can be cleared by simply adding the class clearfix to the float's parent element.

float is best used when you want to put elements on both the left and right edge of a container, and you don't mind if there is a lot of empty space between them. If you're looking to display an evenly spaced row of elements, try using flexbox or inline-block. But if you just want an element on the left and one on the right, float is your best bet. This is particularly common in headers and footers.

float: left
Element 2
float: left
Element 3
float: right
Element 4

Note that multiple adjacent right-floated elements will be displayed in right-to-left order.
