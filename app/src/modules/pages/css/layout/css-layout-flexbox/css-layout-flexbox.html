There are three common ways to cause div elements to display side-by-side: Float , Inline-Block , and flexbox.

Flexbox is still relatively new, and has undergone a lot of changes since it was first implemented. This means that the syntax and relevant CSS properties for flexbox is a mess across browsers - IE9 and lower don't support it at all, and other browsers may only support older versions of the syntax. Furthermore, there are a number of online articles with information on flexbox, but because of all the changes between 2010 and 2013, it is sometimes hard to be sure if what you're reading is still accurate. Combine this with the fact that there are at least 10 CSS properties specifically for flexbox - some of which may have had their names changed or have even ceased to exist over the years - and you've got one seriously confusing setup on your hands. This confusion will tend to clear up over time, as browsers get up-to-date and begin to agree on a more uniform approach to flexbox, but that small comfort won't help you much in the here and now.

For this reason, we're not going to get into the murky details or flexbox here; instead, we're just going to focus on using it in Mean Seed, which isn't nearly so complicated.

The basic idea of flexbox is to create a group of 'flexible boxes' inside some container. The boxes will touch both edges of the container and completely fill the space between, never wrapping to a new line. You can specify their width relative to one another as well, and they will size themselves accordingly, and maintain their ratios no matter how the container resizes. Furthermore, each box will have the same height, filling the container. These properties allow flexbox to easily accomplish a number of tasks that would be very difficult and tedious otherwise, which is why coders have been willing to put up with all of the confusion that has come with it thus far.

To use flexbox in Mean Seed, give the classes flexbox and row to the parent element (you can also use column instead of row, to create a vertical flexbox rather than a horizontal one). Next, specify how flexible each child should be using the classes flex0, flex1, flex2, or flex3. The higher the number, the more space the element will take up relative to its siblings. flex0 elements will not take up any more space than they need; in fact, you can just omit flex0 on elements that should be inflexible.
In theory, you could use any integer, but only these four classes are provided by Mean Seed by default.
