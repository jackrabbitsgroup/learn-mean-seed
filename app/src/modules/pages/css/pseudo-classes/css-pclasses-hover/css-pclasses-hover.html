CSS: The Hover Pseudo-Class
{{pagesFullPath + 'css/pseudo-classes/css-pclasses-hover'}}
The :hover selector is one of the more useful pseudo-classes, even though it doesn't work on touchscreen devices. As you've probably guessed, css rules written under a :hover pseudo-class are only applied when the mouse is hovering over the element in question. This is by the far the cleanest and easiest way to add hover functionality to your website, though it is limited to CSS only - if you need to run javascript code on hover, try something else, like Angular's ng-mouseenter.
