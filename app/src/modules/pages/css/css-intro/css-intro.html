Introduction to CSS

CSS stands for 'Cascading Style Sheets', and its primary purpose is to make your webpage look prettier. CSS often plays a large role in the positioning of elements of your page, and is also used to adjust things like colors, borders, spacing, and sizing. It can even be used to add some limited functionality to your page using things like transitions or hover events.

CSS is written by assigning 'values' to 'properties' under a given 'selector'. The syntax looks like this:

selector1
{
    property1: value1;
    property2: value2;
}

selector2
{
    property3: value3;
    property4: value4;
}

Selectors come in many forms, but their purpose is always to identify some element on the page. Any elements that are identified by a given selector will be assigned the styles specified by the properties and values attached to that selector. This group of styles attached to the selector is sometimes called the selector's 'rule'.

There are many CSS properties to choose from, and each one has its own set of possible values. This can make learning CSS seem a bit daunting at first, but you'll quickly learn all of the common properties and values in the other CSS lessons, and the internet can tell you about any others you have questions about. The CSS Reference at W3Schools is particularly helpful here.

One last thing to note: In Mean Seed, we use a CSS preprocessor - either LESS or SCSS (but not both on the same project) - instead of straight CSS. That means that if you're trying to write CSS in Mean Seed, you should be editing a file with a .less or .scss extension. You don't need to understand LESS or SCSS just yet, though; for now, just know that anything that works in normal CSS will work just the same in LESS or SCSS.
