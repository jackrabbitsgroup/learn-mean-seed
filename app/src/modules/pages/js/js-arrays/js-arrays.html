JS: Arrays
{{pagesFullPath + 'js/js-arrays'}}
Arrays are a useful way of storing data, and should be familiar to experienced coders. An array is a list of 'elements' (pieces of data), in some order. Each element has a specific index in the array; the first element is at index 0. Elements in an array can be of any data type - number, string, function, object, array, etc.

Arrays in Javascript are really just a special type of object that Javascript provides native support for, which is why the syntax is so similar. You don't need to worry about this relationship too much, though, since Javascript handles those details for you.

Javascript provides several methods for dealing with arrays. You can find the full list at W3Schools. The most commonly used (and therefore important to know) methods are indexOf, push, sort, and splice.
