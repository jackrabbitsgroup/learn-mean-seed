AngularJS is the Javascript (MVC) framework we use. It's built on top of Javascript and HTML and handles a lot of the basics for us so coding is much easier!
The main features of Angular are:
1. 2 way data-binding with $scope
2. Dependency Injection and Testing

The main things you'll build in Angular are:
1. routes (controllers)
2. directives - user facing reusable components that use HTML and CSS and touch the "DOM" (Document Object Model)
3. services - javascript only (no HTML or CSS) reusable components/libraries. Often used for "models" - data syncing and storage.
